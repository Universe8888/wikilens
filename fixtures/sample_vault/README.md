# Sample vault (fixture)

This directory will hold a small, synthetic Markdown vault used as the canonical test fixture
for wikilens agents. Populated in P2.

Rules:
- Public-domain or synthetic content only.
- Small (≤ ~50 notes) so tests run fast.
- Deliberately includes broken links, contradictions, and gaps so agents have something to find.

Real / private vault content must NEVER land here. Use `fixtures/private/` locally
(gitignored) for experimentation.
